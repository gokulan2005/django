About Django

What is Django?

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design.

Key Features of Django:

  • Follows the DRY (Don't Repeat Yourself) principle
  • Includes an ORM (Object-Relational Mapping) system
  • Built-in admin interface for easy content management
  • Scalable and secure

Django Commands:

Command Description
django-admin startproject project_name Creates a new Django project
python manage.py runserver Starts the development server
python manage.py startapp app_name Creates a new Django app within the project
python manage.py makemigrations Creates migration files based on changes to models
python manage.py migrate Applies database migrations
